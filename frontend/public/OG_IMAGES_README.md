# Open Graph Image Guidelines

This folder should contain Open Graph (OG) images for social media sharing.

## Required Images

1. **og-image.png** (1200x630px) - Default/homepage
2. **og-courses.png** (1200x630px) - Courses listing page
3. **og-skills.png** (1200x630px) - Skills management page

## Image Specifications

- **Size**: 1200 x 630 pixels (Facebook & LinkedIn recommended)
- **Format**: PNG or JPG (PNG preferred for quality)
- **Max file size**: < 1MB (smaller is better for loading speed)
- **Aspect ratio**: 1.91:1

## Design Guidelines

- Include SkillHub branding/logo
- Use gradient backgrounds matching the app theme
- Keep text large and readable (min 40px font size)
- Ensure key information is in the safe zone (center 1000x500px)
- Use high contrast for readability
- Test how it looks in Facebook, Twitter, and LinkedIn previews

## Tools for Creating OG Images

- Figma (recommended)
- Canva
- Adobe Photoshop
- Online generators: https://www.opengraph.xyz/

## Testing

Test your OG images using:

- Facebook Sharing Debugger: https://developers.facebook.com/tools/debug/
- Twitter Card Validator: https://cards-dev.twitter.com/validator
- LinkedIn Post Inspector: https://www.linkedin.com/post-inspector/
